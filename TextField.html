Prefixes & Suffixes

Ya habíamos visto los iconos de prefijos y no sufren cambio alguno

account_circle
phone

Si necesitas un icono o texto complemetario agrega la clase ".sufix" al contenedor del input

lbs
cancel 
                            
                                <!--Prefixes-->
                                <div class="input-field col s12 m6">
                                    <i class="material-icons prefix">account_circle</i>
                                    <input id="icon_prefix" type="text" class="validate">
                                    <label for="icon_prefix">First Name</label>
                                </div>
                                <div class="input-field outlined col s12 m6">
                                    <i class="material-icons prefix">phone</i>
                                    <input id="icon_telephone" type="tel" class="validate">
                                    <label for="icon_telephone">Telephone</label>
                                </div>

                                <!--Suffixes-->
                                <div class="input-field sufix col s12 m6">
                                    <input id="icon_prefix" type="text" class="validate">
                                    <label for="icon_prefix">Label</label>
                                    <span class="sufix-text">lbs</span>
                                </div>
                                <div class="input-field outlined sufix col s12 m6">
                                    <input id="icon_telephone" type="tel" class="validate">
                                    <label for="icon_telephone">Label</label>
                                    <i class="material-icons">cancel</i>
                                </div>

Dense text field

Si no quieres agregar un label agregar la clase "dense" y el largo se modificará a 42px

                            
                                <div class="input-field col s12 m6">
                                    <input id="last_name" type="text" placeholder="Placeholder" class="validate dense">
                                </div>
                                <div class="input-field outlined col s12 m6">
                                    <input id="last_name" type="text" placeholder="Placeholder" class="validate dense">
                                </div>

Character Counter

Usalo para restringir los carácteres maximos

                            
                                <div class="input-field col s12 m6">
                                    <input id="input_text" type="text" data-length="10">
                                    <label for="input_text">Input text</label>
                                </div>
                                <div class="input-field outlined col s12 m6">
                                    <textarea id="textarea2" class="materialize-textarea" data-length="120"></textarea>
                                    <label for="textarea2">Textarea</label>
                                </div>

Inicialización

                                    
                                        $(document).ready(function() {
                                            $('input#input_text, textarea#textarea2').characterCounter();
                                        });

Changing colors

Cambia los colores del input outlined

                                    
                                        /* label color */
                                        .input-field label {
                                            color: #000;
                                        }
                                        /* label focus color */
                                        .input-field input:focus + label {
                                            color: #000 !important;
                                        }
                                        /* label underline focus color */
                                        .input-field input:focus {
                                            border: 2px solid #000 !important;
                                            -webkit-box-shadow: 0 0px 1px 0 #000 !important;
                                                    box-shadow: 0 0px 1px 0 #000 !important;
                                        }
                                        /* valid color */
                                        .input-field input.valid {
                                            border: 2px solid #000 !important;
                                            -webkit-box-shadow: 0 0px 1px 0 #000 !important;
                                                    box-shadow: 0 0px 1px 0 #000 !important;
                                        }
                                        /* invalid color */
                                        .input-field input.invalid {
                                            border: 2px solid #000 !important;
                                            -webkit-box-shadow: 0 0px 1px 0 #000 !important;
                                                    box-shadow: 0 0px 1px 0 #000 !important;
                                        }
                                        /* icon prefix focus color */
                                        .input-field .prefix.active {
                                            color: #000 !important;
                                        }
                                        /* icon sufix color */
                                        .input-field.sufix i {
                                            color: #000 !important;
                                        }
                                        /* text sufix color */
                                        .input-field.sufix .sufix-text {
                                            color: #000 !important;
                                        }
